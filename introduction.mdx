## Get up and running in three steps

New to Olira? Follow these steps to go from zero to your first successful API call.

<Steps>
  <Step title="Create an account and get your API key">
    Sign up for a free account at [app.olira.ai](https://app.olira.ai). Once you're in, navigate to **Settings → API Keys** to generate your first API key. Copy it somewhere safe — you'll need it for every request.
  </Step>
  <Step title="Make your first API call">
    With your API key in hand, send a request to the Olira REST API at `https://api.olira.ai/v1`. Pass your key as a Bearer token in the `Authorization` header. Check out the [Quickstart](/quickstart) for a ready-to-run example.
  </Step>
  <Step title="Explore the full API">
    Once your first call succeeds, dive deeper. Browse the [API Reference](/api-reference/introduction) to discover all available endpoints, or head to the [SDKs](/sdks/overview) page to integrate Olira using your preferred language.
  </Step>
</Steps>
